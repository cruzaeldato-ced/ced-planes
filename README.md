# ⚡ CED · Commercial Planner 2026

An interactive quoting and planning tool for digital media campaigns. It allows sales teams and clients to visualize, build, and calculate the cost of content campaigns (news, carousels, memes, videos, and stories) with automatic volume discounts applied in real-time.

## 🚀 Key Features

The application is divided into three main sales modules:

* 📰 **News Plan:** Tiered subscription packages (ranging from "Basic Impact" to "Total Domination") focused on brand exposure through daily news coverage.
* 🛒 **Individual Matrix:** A "shopping cart" style quoter to build custom campaigns piece by piece, featuring a progress bar that incentivizes reaching the next discount tier.
* ⚡ **Comprehensive Plan:** An advanced planner that mixes all available formats using preset "Combos," automatically calculating estimated organic reach and quarterly financial projections.

## 🛠️ Tech Stack (Zero-Build Setup)

This project is designed to be ultra-portable. It requires no Node.js, Webpack, or build steps to run in development or production.

* **React 18 & ReactDOM** (via CDN)
* **Babel Standalone** (in-browser JSX compilation)
* **HTML5 & CSS3** (Inline styles and native variables)

## 📦 Installation & Usage

Literally, all you need is a web browser.
